## **Controls**

### Xbox controller

Command         | Action
-------         | ------
Left Stick      | Move / Aim
Start           | Pause menu
Right Trigger   | Shoot
A               | Jump
B               | Teleport (change plan)

### Keyboard (QWERTY)

Command               | Action
-------               | ------
R                     | Reload scene
T                     | Decrease delay between teleportation
K                     | Kill all enemies
C                     | Kill visible enemies
Right arrow / D       | Move right
Left arrow / A        | Move left
W                     | Teleport (change plan)
F                     | Shoot
Space bar             | Jump
